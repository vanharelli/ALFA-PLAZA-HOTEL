# Alfa Plaza Hotel - Ficha Cadastral Digital

Este repositório contém o módulo de Ficha Nacional de Registro de Hóspedes (FNRH) Digital do Alfa Plaza Hotel.

## Funcionalidades
- Check-in Digital via Interface Web
- Integração com WhatsApp para envio de dados
- Conformidade com LGPD e FNRH
- Tecnologia Stateless (Zero Armazenamento de Dados Pessoais)
